# ScrapyPractice

## English

Personal scrapy projects for practicing how to scrape websites using scrapy.

Scrapy is python based open source framework for extracting the data you need from websites.

All of my scrapy projects in this repository obey [robots exclusion standard](https://en.wikipedia.org/wiki/Robots_exclusion_standard). I am also trying to make my projects not violate any laws and rights.

If you found any violation of laws or other's legal right in this repository, please let me know by creating an issue or emailing to me(chadaeun57@gmail.com). Advice for other problems are also welcome!

You can install scrapy by following insturctions in [Installation guide](https://doc.scrapy.org/en/latest/intro/install.html)

When you clone or download this repository and run my scrapy projects, please do not forget to modify `USER_AGENT` value in `settings.py` for each projects. This value is set to 'hohner64'(my most-use nickname) which represents me, so you have to change this value to represent yourself for target website.

References:

- [Scrapy](https://scrapy.org/)
- [Scrapy documentation](https://docs.scrapy.org/en/latest/)

## 한국어

Scrapy를 활용한 웹사이트 스크래핑 연습을 위한 개인 scrapy 프로젝트들입니다.

Scrapy는 웹사이트에서 원하는 정보를 추출하기 위한 파이썬 기반 오픈소스 프레임워크입니다.

모든 프로젝트는 [로봇 배제 표준](https://ko.wikipedia.org/wiki/%EB%A1%9C%EB%B4%87_%EB%B0%B0%EC%A0%9C_%ED%91%9C%EC%A4%80)을 준수하고 있으며, 모든 프로젝트들이 그 어떤 법이나 권리도 침해하지 않도록 만들고자 합니다.

만약 포함된 프로젝트가 법률이나 법적 권리를 침해하고 있다면, 이 리포지토리에 대한 이슈를 만들거나 chadaeun57@gmail.com으로 이메일을 보내 알려주시면 감사하겠습니다. 다른 문제에 대한 조언도 환영입니다!

[Installation guide](https://doc.scrapy.org/en/latest/intro/install.html)의 지시 사항을 따라 scrapy를 설치할 수 있습니다.

이 리포지토리를 클론하거나 다운로드한 뒤 scrapy 프로젝트를 실행할 때, 각 프로젝트의 `settings.py`의 `USER_AGENT` 값을 변경해주시길 바랍니다. 이 값은 저를 나타내는 'hohner64'(제가 가장 많이 쓰는 닉네임입니다)로 설정되어 있기 때문에, 대상 웹사이트를 위해 여러분 자신을 나타내는 값으로 변경해주시기 바랍니다.
